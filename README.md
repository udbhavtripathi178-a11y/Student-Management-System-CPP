# 🎓 Student Management System (C++)

A console-based Student Management System developed in C++. This project was built to practice core C++ programming concepts such as structures, arrays, functions, loops, searching, updating, deleting, and input validation.

---

## ✨ Features

- ➕ Add Student
- 📋 Display All Students
- 🔍 Search Student by Roll Number
- ✏️ Update Student Details
- 🗑️ Delete Student Record
- 🚫 Prevent Duplicate Roll Numbers
- 📊 Display All Student Records
- ✅ Dynamic Student Count
- ✅ Menu-Driven Interface
- ✅ Input Validation

## 🚧 Challenges I Faced During Development

While building this project, I faced several problems and solved them one by one:

- Display function printed all 500 array elements instead of only entered students.
- Understanding pass-by-reference (int &j) and why *j is not used.
- Detecting duplicate roll numbers.
- Printing structure data correctly.
- Understanding getline() and why it skips input.
- Learning about the input buffer.
- Understanding the purpose of cin.ignore().
- Understanding the purpose of cin.clear().
- Fixing infinite loops caused by invalid input.
- Managing dynamic student count.
- Keeping previously entered records while adding new students.
- Implementing Search functionality.
- Implementing Update functionality.
- Implementing Delete functionality by shifting array elements.
- Preventing invalid menu input.
- Handling incorrect user input such as alphabets in integer fields.
- Checking whether the student database is empty before Display, Search, Update, or Delete operations.
